# QR Code Generator

Это веб-приложение для генерации QR-кодов из URL-адресов с дизайном в стиле OZON.

## Функциональность

- Ввод URL-адреса
- Генерация QR-кода
- Возможность скачивания QR-кода как PNG-изображения
- Адаптивный дизайн

## Технологии

- HTML
- CSS
- JavaScript
- Используется библиотека qrcode.min.js для генерации QR-кодов

## Дизайн

Стилизация выполнена в цветовой схеме OZON (оранжевый #ff9f00, черный, белый).
